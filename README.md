# parkinson-disease
This model predicts whether a person has parkinson disease or not using biomedical voice measure
Tech Stack:
Python
Pandas, NumPy
Scikit-learn
Logistic Regression / ML Classifier
Workflow:
Load Parkinson’s dataset

Feature selection & preprocessing
Train-test split
Model training
Prediction & evaluation

Use Case:
Early detection of Parkinson’s disease using non-invasive voice data.
